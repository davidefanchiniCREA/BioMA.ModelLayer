# TestsOutputToListener class

Writes the output of pre/post conditions tests to the [`CRA.ModelLayer.TraceHelper`](../CRA.ModelLayer/TraceHelper.md) class. Then the user can attach a listener to this trace source, to output the log. The log level used is System.Diagnostic.TraceEventType.Warning. The event id is 1000.
