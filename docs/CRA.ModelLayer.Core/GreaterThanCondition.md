# GreaterThanCondition class

Condition to test that one VarInfo value is greater than another. This kind of condition needs two VarInfos to work. It checks that the first VarInfo is greater than the second VarInfo. If the opposite happens, it gives errror. It is an implementation of [`ICondition`](./ICondition.md) interface.
