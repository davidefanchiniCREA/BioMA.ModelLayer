# RangeOneRangeTwoCondition class

Condition that works on two VarInfo objects: it tests that if one VarInfo value is inside its validity range, even a second VarInfo value is inside its own range. If the second is inside the range, while the first outside the range, it gives error since the condition is not satisfied. In all the other cases the condition is satisfied. To be in the validity range means that a VarInfo value is greater than the VarInfo minimum value and less than the VarInfo maximum value. It is an implementation of [`ICondition`](./ICondition.md) interface.
