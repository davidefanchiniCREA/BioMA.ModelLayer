# ConditionsCollection.AddCondition method

Add a condition to the collection. The method checks if the condition is applicable, through the IsApplicable method. If not, it throws exception. The method checks if the condition is already present in the collection. If yes, it throws an exeception. The condition is registered among the other conditions having the same Type.
