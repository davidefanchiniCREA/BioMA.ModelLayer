# NotZeroIfSecondNotZeroCondition class

Condition that works on two VarInfo objects: it tests that the first VarInfo value is different from zero, when the second VarInfo value is different from zero. If the second is different from zero, while the first is equal to zero, it gives error since the condition is not satisfied. In all the other cases the condition is satisfied. It is an implementation of [`ICondition`](./ICondition.md) interface.
