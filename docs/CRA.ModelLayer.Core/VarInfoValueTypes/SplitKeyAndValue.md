# VarInfoValueTypes.SplitKeyAndValue method

To be used when parsing dictionary-like entries, having a key and a value. Given one of these entries as the parameter 'complexValue', in the form of a unique string that is the concatenation of the value, the separating character "$" and the key, returns the value and populates the output parameter 'key' with the key.

```csharp
public static string SplitKeyAndValue(string complexValue, out string key)
```

| parameter | description |
| --- | --- |
| complexValue | The entry to be parsed in the form "value" + "$" + "key". |
| key | After the invocation, contains the "key". |

## Return Value

The "value".
