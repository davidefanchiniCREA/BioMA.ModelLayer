# IVarInfoConverter interface

Manages the logic for parsing string or XML representation of the [`VarInfo.CurrentValue`](./VarInfo/CurrentValue.md) to a correctly parsed value and back. An implementation of this interface has to be written for each parsed type of a [`VarInfo.CurrentValue`](./VarInfo/CurrentValue.md), precisely for each of the [`VarInfoValueTypes`](./VarInfoValueTypes.md) returned by [`VarInfoValueTypes.Values`](./VarInfoValueTypes/Values.md) Since this interface is used also as a central point in parsing/serializing [`VarInfo`](./VarInfo.md) values in the MPE application (that has peculiar formats for serialization), specific methods are present, namely:

* **Method** – Description
* **[`GetParsedValue(XElement)`](./IVarInfoConverter/GetParsedValue.md)** – Parses to the correct type the [`VarInfo.CurrentValue`](./VarInfo/CurrentValue.md) XML representation peculiar to the MPE application.
* **[`ParseValue(object, string)`](./IVarInfoConverter/ParseValue.md)** – Serializes the [`VarInfo.CurrentValue`](./VarInfo/CurrentValue.md) parsed in its correct type to the XML representation peculiar to the MPE application for the type managed by this instance.
* **[`ParseValueForMPE(object)`](./IVarInfoConverter/ParseValueForMPE.md)** – Serializes the [`VarInfo.CurrentValue`](./VarInfo/CurrentValue.md) parsed in its correct type to the List&lt;string&gt; representation peculiar to the MPE application for the type managed by this instance.
* **[`GetParsedValueFromMPE(List&lt;string&gt;)`](./IVarInfoConverter/GetParsedValueFromMPE.md)** – Parses to the correct type the [`VarInfo.CurrentValue`](./VarInfo/CurrentValue.md) List&lt;string&gt; representation peculiar to the MPE application.
