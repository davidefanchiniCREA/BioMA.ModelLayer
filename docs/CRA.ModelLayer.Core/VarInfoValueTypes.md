# VarInfoValueTypes class

[Typesafe Enum](https://www.infoworld.com/article/3198453/how-to-implement-a-type-safe-enum-pattern-in-c.html) having an instance for each parsed type that a [`VarInfo`](./VarInfo.md) can take. It provides parsing functionalities for converting from string representations of [`VarInfo.CurrentValue`](./VarInfo/CurrentValue.md) to parsed values and back. The constructor is private and the only available instances of this class can be obtained from public static properties (a variation of the [Singleton pattern](http://en.wikipedia.org/wiki/Singleton_pattern), where instead of an instance there is a set of instances). Each instance of [`VarInfo`](./VarInfo.md) is configured with an object of this type by setting its [`ValueType`](./VarInfo/ValueType.md) property to an instance of this class.

## Remarks

[`VarInfo`](./VarInfo.md)'s [`CurrentValue`](./VarInfo/CurrentValue.md) types are implemented this way to have a single point of maintenance (this class) for parsing functionalities and to avoid cumbersome `switch` statements in applications dealing with [`VarInfo`](./VarInfo.md).
