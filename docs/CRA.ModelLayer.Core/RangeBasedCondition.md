# RangeBasedCondition class

Condition to test that a VarInfo value is greater than the VarInfo minimum value and less than the VarInfo maximum value. If the VarInfo is outside the range, it gives error since the condition is not satisfied. In all the other cases the condition is satisfied. It is an implementation of [`ICondition`](./ICondition.md) interface.
